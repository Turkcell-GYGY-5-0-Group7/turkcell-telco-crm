# Workflow: Add Feature

## Trigger

* new task in sprint
* Product Owner or Tech Lead request

---

## Phase 1 — Impact Analysis

Architecture Agent:

* identifies service mode
* validates change scope

Domain Engineer:

* proposes implementation approach

---

## Phase 2 — Design Proposal

Generate:

* feature design
* affected modules
* event changes (if any)

---

## Phase 3 — Domain Implementation Plan

Domain Engineer:

* CQRS commands/queries
* service updates
* validation rules

---

## Phase 4 — Event Check

Event Integration Agent:

* determines event impact
* versioning check

---

## Phase 5 — Output

* implementation plan
* risk analysis
* approval request
